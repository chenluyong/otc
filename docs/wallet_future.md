# 正常流程
用户流程
1. 选择商家，输入交易额额度
2. 查看并选择付款方式，进行线下打款
3. 打款后，点击【确认支付】等待商家打币
4. 收到数字货币，提现到链上

商家流程
1. 商家转入USDT至平台
2. 商家设置收款方式、收款二维码、交易额度区间、商家留言
3. 点击【开始营业】，设置营业时间
4. 商家接到订单，冻结部分交易金额
5. 商家收到【确认支付】的信息，确认银行卡是否有款项到达
6. 确认打款后，放行数字货币
